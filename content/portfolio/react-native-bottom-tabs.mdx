### About

React Native Bottom Tabs brings truly native tab bar components to React Native apps. Unlike JavaScript-based solutions, this library uses native platform primitives for the best performance and authentic platform feel.

## Supported Platforms

- iOS
- Android
- iPadOS
- visionOS
- tvOS
- macOS

## Features

- Native platform primitives for authentic look and feel
- Seamless integration with React Navigation
- Support for SF Symbols on iOS and Material Icons on Android
- Optimized performance with native rendering
- Full platform SDK access

### About

React Native visionOS allows you to write visionOS apps with full support for the platform SDK. This is a full fork of React Native with changes needed to support Apple's spatial computing platform.

## Getting Started

You can quickly initialize a new visionOS project:

```sh
npx @callstack/react-native-visionos@latest init YourApp
```

## Features

- Full visionOS platform support
- Native spatial UI components
- Integration with RealityKit
- Support for immersive spaces
- Hand tracking and eye tracking support
- Full access to visionOS SDK
